Local install​

  1. Download nodejs latest from https://nodejs.org/en/download

  2. Add to $PATH 

    export PATH=$PATH/{YOUR DOWNLOAD PATH}/bin

  3. Install Yarn via 

    npm install yarn -g

  4. Install Default dependence

    yarn

  5. Simply run blog

    yarn start

Add new article​

  1. Create your markdown file in docs , if your file named "test.md" 

    touch /docs/test.md

    In order to achieve classification purposes, new folders can be created when necessary, in /docs

    touch /docs/type_A/test.md

  2. Write code & article

    We are supposed to change value in {value} .


      ---
      description: {example}
      title: {example}
      tags:
        - {tag1}
        - {tag2}
        - {tag3}
      keywords:
        - {tag1}
        - {tag2}
        - {tag3}
      image: {https://github.com/tianrking.png}
      last_update:
        date: {12/6/2022}
        author: {w0x7ce}
      ---

      ## P1
      ## P2

  3. Test code 

    yarn start

  4. Build Website

    yarn build

  5. Develop to Github & Server

    git add .
    git commit -m {message}
    git push -u {remote} {branch}

Github Maintenance​

  1. Generating a new SSH key & Make sure SSH key exist

    ssh-keygen -t ed25519 -C "your_email@example.com"

  • Public key

    cat ~/.ssh/{id_ed25519.pub}

  • Private key

    cat ~/.ssh/{id_ed25519}

  1. Add your Public key in Github Personal Keys

    Click -> {New SSH key} -> Paste {Public key}

  2. Add your Private Key in [Repo Secrets For Actions] for Github Actions

    Click {New repository secret} -> Paste {Private key}

    https://github.com/{USER}/{REPO}/settings/secrets/actions
